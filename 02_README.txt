# Personal Portfolio Page

This is a simple HTML + CSS project that creates a basic personal portfolio website.

## Features:
- Header with navigation links
- About Me section
- Skills section
- Projects section
- Contact information
- Footer with copyright

## Files:
- 01_index.html → Main HTML file
- 01_style.css → CSS styling file

## How to Run:
1. Open the folder in VS Code.
2. Open index.html in a browser (Right-click → Open with Browser).
3. You will see the portfolio webpage.
