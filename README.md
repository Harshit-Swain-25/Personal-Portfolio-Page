# Personal Portfolio Website

A simple personal portfolio website built using HTML and CSS. This project showcases personal information, technical skills, projects, and contact details in a clean and organized layout.

## 📌 Project Overview

This portfolio website is designed to introduce the developer and present their basic information in a professional manner.

The website contains multiple sections that allow visitors to quickly learn about the developer's background, skills, projects, and contact information.

This project is suitable for beginners who want to practice fundamental web development concepts and create their first portfolio website.

## 🚀 Features

* Simple and clean user interface
* Navigation menu for easy access to different sections
* About Me section
* Skills section
* Projects section
* Contact section
* Responsive page structure
* Beginner-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3

## ▶️ How to Run the Project

1. Download or clone this repository.

```bash
git clone https://github.com/Harshit-Swain-25/Personal-Portfolio-Page.git
```

2. Open the project folder.

3. Open `index.html` in any web browser.

No installation, dependencies, or additional software are required.
